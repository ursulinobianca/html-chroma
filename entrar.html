1. Informações gerais
A Política de Privacidade contém informações a respeito do modo como tratamos, total ou parcialmente, de forma automatizada ou não, os dados pessoais dos usuários que acessam nosso aplicativo. Seu objetivo é esclarecer os interessados acerca dos tipos de dados que são coletados, dos motivos da coleta e da forma como o usuário poderá atualizar, gerenciar ou excluir estas informações. Essa política foi elaborada conforme a Lei Federal nº 13.709/2018 (Lei de Proteção de Dados Pessoais).

2. Direitos do usuário
O aplicativo se compromete a cumprir as normas previstas no RGPD (Regulamento Geral de Proteção de Dados), em respeito aos seguintes princípios:
Os dados do usuário serão processados de forma lícita e transparente. Os dados pessoais do usuário serão coletados apenas para finalidades determinadas, explícitas e legítimas, não podendo ser tratados posteriormente de uma forma incompatível com essas finalidades. Os dados pessoais do usuário serão conservados de uma forma que permita a identificação dos titulares dos dados apenas durante o período necessário para as finalidades para as quais são tratados. Os dados pessoais do usuário serão tratados de forma segura, protegidos do tratamento não autorizados. Os dados pessoais do usuário serão apagados assim que ele se desligar do dispositivo.

3. Política de dados
O que coletaremos Coletaremos dados de identificação do usuário, para a realização do cadastro e ordem em que acessou a exposição. Os seguintes dados serão coletados:
Nome;
Ordem das obras acessadas.

O que não coletaremos:
Não serão coletados dados sensíveis do usuário, dados definidos nos artigos 9º e 10ª do RGPD e nos artigos 11ª e seguintes da Lei de Proteção de Dados Pessoais. Assim os seguintes dados não serão coletados: Dados que revelem a origem racial ou étnica, as opiniões políticas, as convicções religiosas ou filosóficas, ou a filiação sindical dos usuários; Dados genéticos;
Dados biométricos para identificar uma pessoa de forma inequívoca;
Dados relativos à saúde do usuário;
Dados relativos à vida sexual ou à orientação sexual do usuário;
Dados relacionados a condenações penais ou a infrações ou com medidas de segurança conexas.

4. Não fornecer dados de terceiros
Durante o uso do aplicativo, a fim de proteger os direitos de terceiros, o usuário deverá fornecer apenas seus dados pessoais e não o de terceiros.

5. Medidas de segurança
Levamos a sua privacidade a sério e tomamos todas as medidas e precauções para proteger seus dados. Temos medidas de segurança incluindo: criptografia, restrição de acesso, firewalls, etc. No entanto, o aplicativo se exime de reponsabilidade por culpa exclusiva de terceiros, como em caso de ataques de hackers ou crackers, ou culpa exclusiva do usuário, como no caso em que ele mesmo transfere seus dados a um terceiro.

6. Autorizações
Esse aplicativo necessita da autorização para o uso do autofalante (para o uso de fones de ouvido), da câmera (para a leitura do QR code) e do Bluetooth (para a conexão)
